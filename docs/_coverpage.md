# tMind

> 基于 tFrameV9 的全新平台

* 前端路线：vue3、vue-router、axios、vuex
* 后端路线：koa、
* 全局依赖：tMind-core、tMind-...
* 环境要求：nodeJs(v14.15+)，CentOS(v7 or 其他linux发行版)
* 部署方式：local or cloud (阿里云、华为云)
* UI类库：ElementPlus
* DB支持：PostgreSQL（首选）、MySQL
* WEB服务：Nginx
* 浏览器支持：Chrome（首选）、Firefox、Opera、其他国产浏览器新版
* 浏览器约束：！不支持IE系列 ！
